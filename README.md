# React Environment 
- 請先確認電腦有安裝 node.js 
---
- 確定版本(確認有安裝node.js)

```
node --version
```

指令

``` 
npx create-react-app my-app
```

- 安裝後約有200MB
- 下載後，執行以下指令
```
cd my-app
npm start
```
之後會在預設瀏覽器開啟網頁，即完成環境架設。

- 所有要編輯的都方在 src 裡面，其他資料夾盡量不要動。

- 要寫的 html 會放在 App.js 的 div 標籤裡面，div container 純粹用來包住 html ，沒有其他用途。
